# @liberte-top/shared

`@liberte-top/shared` is the first package extracted into `ts-packages`.

## Scope

- auth helpers for session snapshot refresh and login redirect handling
- `defer` for acquire-run-release flows
- `ensure` for explicit invariant checks
- `openapi` fetch helpers for JSON responses
