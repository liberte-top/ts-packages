# @liberte-top/components

`@liberte-top/components` is the shared Svelte component package for `liberte-top` web surfaces.

## Scope

- source-first Svelte 5 components for app and service UIs
- token-driven theme defaults exposed through `theme.css`
- small, explicit primitives rather than a large design-system surface
